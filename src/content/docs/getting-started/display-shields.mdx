title: Display Shield Overview
description: Learn about the display shields for the BBC micro:bit
---

There are a variety of display shields available for the BBC micro:bit 
that enhance its capabilities by providing a color display and additional
buttons. The microbit slots into the shield using its edge connector.

## Small screen shields

* [Kittenbot's Newbit Arcade shield](https://www.kittenbot.cc/products/newbit-arcade-shield)
* [ELECFREAK's micro:bit Arcade shield](https://shop.elecfreaks.com/products/micro-bit-retro-programming-arcade)
* [Kitronik's Arcade for BBC micro:bit](https://kitronik.co.uk/products/56116-kitronik-arcade-for-bbc-micro-bit-makecode-arcade)
