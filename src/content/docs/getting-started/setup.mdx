Follow these steps to set up your micro:bit for use with our applications.

## Hardware Setup

1. **Connect your micro:bit**: Use a USB cable to connect your micro:bit to your computer
2. **Install drivers**: Your operating system should automatically recognize the device
3. **Verify connection**: Your micro:bit should appear as a mounted drive

## Software Setup

1. Visit [https://makecode.microbit.org](https://makecode.microbit.org)
2. Create or open a project
3. Write your code or use block-based programming
4. Download the `.hex` file to your computer
5. Drag and drop the file onto your micro:bit drive

## Using Our Applications

Each of our applications has specific setup instructions. Check the individual app documentation for details on:
- Required hardware components
- Installation steps
- Getting started guides

## Troubleshooting

If you're having issues:
- Make sure your USB cable supports data transfer (some cables only charge)
- Try using a different USB port
- Check that your micro:bit firmware is up to date
- Visit the [official support page](https://support.microbit.org)

## Next Steps

Once your device is set up, explore our featured applications and start building amazing projects!
